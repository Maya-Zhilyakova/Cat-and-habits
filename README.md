**Кот и привычки** — это мини-виджет для рабочего стола Windows, который помогает формировать полезные привычки. Милый кот-компаньон всегда под рукой, мотивирует вас каждый день, а встроенная статистика помогает не сбиваться с пути.

🎯 **21 день** — и новая привычка станет частью вашей жизни!

Виджет живёт прямо на рабочем столе, не мешает работе и всегда напоминает о ваших целях.
---

## 📚 Функционал

- 🐱 **Кот-мотиватор** — случайные фразы поддержки в каждом разделе
- ⚙️ **Трекер привычек** — добавьте до 3 привычек и отмечайте их ежедневно
- 🔥 **Система стриков** — отслеживайте непрерывные серии выполнения
- 📊 **Статистика** — процент выполнения, текущий и максимальный стрик
- 🎯 **21-дневный цикл** — научно обоснованный срок для формирования привычки
- 🖱️ **Перетаскивание окна** — перемещайте виджет в любое удобное место на экране
- ♻️ **Сброс данных** — полная очистка базы для нового старта

- Python 3.7 или выше


## 🗄️ Структура БД

### Таблица `habits` — привычки

| Поле | Тип данных | Обязательно | Описание |
| :--- | :--- | :---: | :--- |
| **id** | INTEGER | ✅ | Первичный ключ |
| **habit_name** | TEXT | ✅ | Название привычки |
| **created_date** | DATE | ❌ | Дата создания (по умолчанию — текущая) |

### Таблица `habit_logs` — логи выполнения

| Поле | Тип данных | Обязательно | Описание |
| :--- | :--- | :---: | :--- |
| **id** | INTEGER | ✅ | Первичный ключ |
| **habit_id** | INTEGER | ✅ | Внешний ключ → `habits.id` |
| **date** | DATE | ❌ | Дата отметки (по умолчанию — текущая) |
| **completed** | BOOLEAN | ❌ | 1 — выполнено, 0 — не выполнено (по умолчанию 0) |
| **streak_count** | INTEGER | ❌ | Текущая серия выполнения (по умолчанию 0) |
